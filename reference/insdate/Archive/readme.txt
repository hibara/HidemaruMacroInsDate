日付時刻挿入マクロ（DLL付属拡張書式版）　　　　　　　　　　ver.1.03
======================================================================

■説明
-------------------
秀丸エディタ用の日付時刻挿入マクロです。
動作には「秀丸エディタ」（ver.8 以上）が必要です。これらファイル単体で
は動きません。

秀丸エディタマクロにある「日付と時刻関連キーワード」を使っても、ある程
度の日付文字列を挿入することができますが、和暦（昭和・平成など）とか、
ゼロパディングしない月日（2月や2日など）を指定したいとか、昨日、明日な
ど、現在日時をインクリメント（またはデクリメント）した文字列を挿入した
いときには、マクロだけで処理するには限界が出てきます。

そこで、日時文字列を取得するDLLとセットで、その辺りの書式を自由度高く
設定し、挿入できるようなマクロを作ってみました。

ちなみに、現在編集中のテキストファイル（いま開いている秀丸エディタ）の
タイムスタンプ（更新日時）を取得して挿入する機能もあります。

たとえば、以下のような書式で日時を挿入することができるようになります。

ex).
平成23年4月18日　月曜日
2011/04/18(Mon)
午後 6時36分58秒


■付属ファイルの説明
-------------------
InsertDate.mac           カーソル位置に日付文字列を挿入します。
InsertDateList01.mac     メニューから選んでいろいろな日付文字列を挿入します。
InsertDateList02.mac     上と同じ。サンプル表示を集めたマクロです。
InsertDateList03.mac     編集中ファイルのタイムスタンプから日時取得するサンプル。
insdate.dll              動作に必須のDLLです。上記マクロから呼ばれます。
install.exe              インストーラです。秀丸エディタ本体の場所を自動で
                         探してインストールします。実行時に、必要があれば
                         管理者での実行の確認メッセージが出ます。
readme.txt               この説明書。


■インストール
-------------------
上記の .mac, .dllを秀丸エディタ本体（Hidemaru.exe）のある場所にコピー
します。

手動でのコピーが難しいようなら、付属のinstall.exeを使ってコピーするこ
ともできます。ただし、Windows Vista, 7などの環境によっては管理者権限に
よる実行が必要になることがあります。

コピーが完了しましたら、秀丸エディタを開き、メニューにある
「マクロ(M)」→「マクロ登録(E)...」を選んで、ご自身で設定を行ってくだ
さい。


■使い方
-------------------
●通常の使い方 --
マクロ内で、DLLをロードした後、以下の関数を呼び出します。
引数に与える書式は、後述の「書式」一覧をご覧ください。

dllfuncstr( #dll, "GetDateString", "yyyy/mm/dd(ddd)", 0);
出力結果： 2011/04/18(月)

●英語表記で挿入する --
英語表記にしたい場合は、「GetDateStringEn」関数を使います。

dllfuncstr( #dll, "GetDateStringEn", "yyyy/mm/dd(ddd)", 0);
出力結果： 2011/04/18(Mon)

※最後の引数は、"0"としておかないと、不定な日付が出力されますので
　ご注意ください。

●日付や日時を増減させて挿入する --
現在日時から進めた日付や時刻、あるいは過去の日時を指定したい場合は、
上記の関数に引数を続けて与えます。

順に、「年，月，日，時，分，秒」となっています。

dllfuncstr( #dll, "GetDateString", "yyyy/mm/dd(ddd)", 1, 0, 0, 0, 0, 0 );
出力結果： 2012/04/18(水)

出力結果は「来年」の今日の日付になります（１年インクリメントされています）。

逆に過去の日付を挿入したい場合は、

dllfuncstr( #dll, "GetDateString", "yyyy/mm/dd(ddd)", 0, -6, 0, 0, 0, 0 );
出力結果：2010/10/18(Mon)

出力結果は、半年前（−６ヶ月）の日付となります。

●ファイルのタイムスタンプから日時を取得して挿入する
以下の関数群が用意されています。

・GetCreateDateString       編集中のテキストファイルの作成日時を取得
・GetCreateDateStringEn     上記の英語表記
・GetLastWriteDateString    編集中のテキストファイルの更新日時を取得
・GetLastWriteDateStringEn  上記の英語表記
・GetAccessDateString       編集中のテキストファイルのアクセス日時を取得
・GetAccessDateStringEn     上記の英語表記

たとえば、以下のように "GetLastWriteDateString" 関数を使うと、いま開い
ている編集中のテキストファイルのタイムスタンプ（更新日時）を取得して挿
入します。

dllfuncstr( #dll, "GetLastWriteDateString", filename, "yyyy/mm/dd(ddd)", 0);
出力結果： 2011/04/18(月) 
　　　　　↑これは現在日時ではなく編集中ファイルの更新日時が出力されます。

※なお、第一引数には、秀丸マクロのファイルパスを参照するキーワード
　「filename」を必ず指定してください。これを誤って指定すると予期せぬ
　動作になります。


■書式
-------------------
指定できる書式は以下の通りです。

d      日付を先頭のゼロなしで表示（1 から 31 まで）
dd     日付を先頭のゼロも含めて表示（01 から 31 まで）
ddd    曜日を省略形（Sun-Sat or 日〜土）で表示
dddd   曜日を完全な形（Sunday-Saturday or 日曜日〜土曜日）で表示
e      和暦を先頭にゼロのついていない数字で表示
ee     和暦を先頭にゼロのついた数字で表示
g      元号の省略形
gg     元号の完全な形
m      月を先頭にゼロのない数字（1-12）として表示
       hまたはhh指定子の直後にm指定子を指定すると、月ではなく分表示
mm     月を先頭にゼロのある数字（01-12）として月を表示
       hまたはhh指定子の直後にmm指定子を指定すると、月ではなく分が表示
mmm    月を省略形（Jan-Dec or 1〜12）で表示
mmmm   月を完全な形（January-December, 1月〜12月）で表示
yy     年を2桁の数字（00-99）で表示
yyyy   年を4桁の数字（0000-9999）で表示
h      時を先頭にゼロのない数字（0-23）として表示
hh     時を先頭にゼロのある数字（00-23）として表示
n      分を先頭にゼロのない数字（0-59）として表示
nn     分を先頭にゼロのある数字（00-59）として表示
s      秒を先頭にゼロのない数字（0-59）として表示
ss     秒を先頭にゼロのある数字（00-59）として表示
z      ミリ秒を先頭のゼロなしで表示（0 〜 999）
zzz    ミリ秒を先頭のゼロ付きで表示（000 〜 999）
am/pm  am/pm を表示
a/p    a/p を表示
ampm   午前/午後 を表示（'hh', 'h'を指定したときのみ12時は0時と表示されます）

なお、上記の書式をそのまま文字として表示したい場合は、シングルクォート
で囲えば、そのまま表示されます。

ex). 'dd'
